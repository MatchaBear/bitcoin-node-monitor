# 🟠 Bitcoin Monitor - Simple Guide

## 🚀 Super Easy Setup (One-Time Only)

1. **Install shortcuts** (makes everything easier):
   ```bash
   ./install-shortcuts.sh
   ```

2. **Restart your terminal** or run:
   ```bash
   source ~/.zshrc
   ```

3. **That's it!** Now you can use Bitcoin Monitor from anywhere! 🎉

## 📱 Simple Commands (Like Using an App)

### 🧪 **Test Everything**
```bash
btc test
```
*Tests if everything is working. Always run this first!*

### 🌐 **Check Your Bitcoin Connections** 
```bash
btc peers
```
*Shows where your Bitcoin peers are connecting from (with country flags!)*

### 📊 **Full Dashboard**
```bash
btc monitor
```
*Complete Bitcoin monitoring dashboard with prices, stats, and more*

### ⚡ **Quick Status Check**
```bash
btc status
```
*Check if everything is installed and working properly*

### 💡 **Get Help**
```bash
btc help
```
*Show all available commands and how to use them*

### 📋 **Version Info**
```bash
btc version
```
*Show what version you're running and features*

## 🎯 Quick Shortcuts (Even Faster!)

For super quick access, you can also use:

- `btc-test` - Quick test
- `btc-peers` - Quick peer check  
- `btc-monitor` - Quick monitoring
- `btc-status` - Quick status
- `btc-help` - Quick help

## 🏃‍♂️ Quick Start Guide

1. **First run**: `btc test`
2. **Check connections**: `btc peers` 
3. **Full monitoring**: `btc monitor`

## 🆘 If Something Goes Wrong

1. **Check status**: `btc status`
2. **Test again**: `btc test`
3. **Get help**: `btc help`

## 💻 What Each Command Does

| Command | What It Does | When To Use |
|---------|-------------|-------------|
| `btc test` | Tests all systems | First time, or when troubleshooting |
| `btc peers` | Shows network connections | Check where Bitcoin peers connect from |
| `btc monitor` | Full dashboard | Monitor Bitcoin price, node status, etc. |
| `btc status` | System health check | Verify everything is installed correctly |
| `btc help` | Show help menu | When you forget commands |

## 🎉 That's It!

You now have a professional Bitcoin monitoring system that you can use like any other app. Just type `btc` and the command you want!

**Example day-to-day usage:**
```bash
# Morning routine
btc test          # Quick test to make sure everything works
btc peers         # See Bitcoin network connections
btc monitor       # Start monitoring (leave it running)
```

No need to remember complex commands or navigate to folders. Just type `btc` + what you want to do! 🚀
